Tutor Bot Senior Project

SP-203 Agentic Multimodal Study Assistant

CS 4850-01, Spring 2026 Semester


For our Senior Capstone Project, we created a web-based study assistant application. Using Google's Agent Development Kit (ADK) we created agents in Python capable of taking in user input, summarizing the user's topics, providing users with step-by-step solutions to their problems, and artifact creation.

We also created a custom React user interface that utilizes Firebase Authentication and Firestore for user authentication and tutor session storage.

Technology/Tools Used

  • Google ADK
  • Python
  • Firebase Authentication
  • Firestore Database
  • React User Interface
  • Fast API
